Kanocc

Kanocc ain't no compiler-compiler

Kanocc is a ruby framework for syntax recognition.

Some of it's highlights are:

  • Handles any context-free grammar
  • No external grammar files. You define the grammar directly in your rubycode through method calls.
  • Semantics of grammarrules is given as ruby blocks.

I wrote Kanocc mainly as a way to learn myself Ruby, but I think it may prove useful in it's own right. If you want to try it out you can go to the project page, where you'll find downloads, or you may browse the documentation.

If you find bugs, please report them at the tracker. Questions may be asked at the forum.

Kanocc is version-controlled with Git. If you want to check out Kanocc, you must have Git installed. You can then do: 

    git clone git://rubyforge.org/kanocc.git
If you have Kanocc checked out already, you can pull updates with 
   git pull
(you must do this from the root of your local Kanocc copy.)

If you want to submit patches - please do so through the tracker.

Enjoy.
